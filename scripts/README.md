# Stack scripts

Usa lo script in `scripts/stack.ps1` su Windows o `scripts/stack.sh` su macOS/Linux.

Comandi disponibili:

- setup: verifica immagini e builda i container
- up: avvia lo stack
- down: ferma lo stack
- restart: riavvia lo stack
- status: stato dei container
- logs: log in streaming
- backup: backup del database in `backups/`
- monitor: docker stats
